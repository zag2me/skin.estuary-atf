# Description
A simple Estuary Mod keeping all its speed and simplicity but adding some nice touches

# Features
- Slimline TV channel browser
- Mvid square thumbs
- New codec icons
- Wide thumb view (Currently needs artwork beef)
- Music track rating stars in list view
